# DISCORD TOKEN CHECKER 

Công cụ kiểm tra trạng thái Token Discord (Valid, Verified, Locked, Invalid) với tốc độ cao

---

## Hướng dẫn cài đặt và sử dụng

Vui lòng làm theo các bước dưới đây để đảm bảo công cụ hoạt động chính xác:

### Bước 1: Cài đặt các thư viện cần thiết
Mở Terminal (CMD hoặc PowerShell) tại thư mục dự án và chạy lệnh sau để cài đặt các thư viện bổ trợ:

```bash
pip install -r requirements.txt
```

### Bước 2: Bỏ token vào input/tokens.txt (số token cần check) và lưu lại

### Bước 3: Chạy lệnh sau

```bash
py token_checker.py
```
